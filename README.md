# React Job Cards Project 👨‍💼

A beginner‑friendly React project where I learned how to use **props** to pass data between components.  
This app displays job openings as reusable **Card** components.

---

##  Features 🚀
- Built with **React + Vite**
- Uses **props** to pass data from `App.jxs` to `card.jsx`
- Each card shows:
  - Company logo
  - Job title and posting date
  - Tags (e.g., Remote, Contract)
  - Pay per hour
  - Location
- Font Awesome icons for styling

---

## What I Learned  📚
- How to create a reusable component (`Card.jsx`)
- How to pass data using **props**:
  ```jsx
  <Card 
    logo={elem.logo} 
    company={elem.company} 
    date={elem.datePosted} 
    post={elem.post} 
    tags={elem.tags} 
    pay={elem.payPerHour} 
    location={elem.location} 
  />
